# 学习中文 - Aprenda Chinês Mandarim

Uma aplicação web interativa para brasileiros aprenderem chinês mandarim de forma divertida e eficaz.

## Sobre o Projeto

Este projeto foi desenvolvido para facilitar o aprendizado do idioma chinês mandarim para falantes de português brasileiro. A plataforma oferece:

- **Lições Interativas** - Exercícios dinâmicos e envolventes
- **Pronúncia Nativa** - Áudio de falantes nativos
- **Curso Completo** - Do básico ao avançado
- **Sistema de Conquistas** - Gamificação para motivar o aprendizado
- **Aprendizado Cultural** - Contexto histórico e social da China
- **Acesso Mobile** - Aprenda em qualquer dispositivo

## Tecnologias

- [Next.js 15](https://nextjs.org/) - Framework React
- [React 19](https://react.dev/) - Biblioteca UI
- [TypeScript](https://www.typescriptlang.org/) - Tipagem estática
- [Tailwind CSS](https://tailwindcss.com/) - Framework CSS

## Instalação

```bash
# Clone o repositório
git clone https://github.com/anderson-ufrj/learning-chinese.git

# Entre no diretório
cd learning-chinese

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no seu navegador.

## Scripts Disponíveis

| Comando | Descrição |
|---------|-----------|
| `npm run dev` | Inicia servidor de desenvolvimento |
| `npm run build` | Gera build de produção |
| `npm run start` | Inicia servidor de produção |
| `npm run lint` | Executa o linter |
